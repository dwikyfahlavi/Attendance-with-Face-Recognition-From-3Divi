# Sound Assets

This folder contains sound files for the attendance system.

## Usage

The app will play a sound when attendance is successfully recorded.

## Adding Custom Sounds

To add custom success sound:

1. Add your sound file (MP3 format recommended) to this folder
2. Name it `success.mp3` or `beep.mp3`
3. The app will automatically use it

## Fallback

If no custom sound file is found, the app will use the system click sound as a fallback.

## Recommended Sound Files

- **success.mp3** - Main success sound (short beep or chime, 0.5-1 second)
- **beep.mp3** - Alternative beep sound

You can download free sound effects from:
- [Pixabay](https://pixabay.com/sound-effects/)
- [Freesound](https://freesound.org/)
- [Zapsplat](https://www.zapsplat.com/)

Search for: "success beep", "notification sound", or "positive chime"
